# Herobrine.fr - Datapack principal

### Information d'utilisation
Ce datapack est destiné à être utilisé dans le cadre du serveur de jeu de rôle Herobrine.fr. 
Le code est cependant entièrement open source et peut être modifié et redistribué librement.

### Contribuer au projet
Il est possible de contribuer au développement du datapack au travers de pull requests.

Il est recommendé d'utiliser **Visual Studio Code** avec l'extension **Datapack Helper Plus by Spyglass**, afin de bénéficier de l'autocomplétion et du surlignage d'erreur.

Il est également possible d'utiliser des **générateurs** tels que https://misode.github.io (All generators). Assurez-vous cependant de sélectionner la bonne version du jeu.

#### Conventions
Le **datapack** cherche à rester aussi fidèle que possible aux **conventions du jeu** et du serveur : les **namespaces** (collectif_radiant, thurgie, herobrine...) doivent donc être les mêmes que ceux du resourcepack, et les fichiers doivent suivre la convention de nommage du jeu. (ex: herobrine:rangerboussole -> herobrine:sort_compass).

Les **commentaires** au sein des .mcfunction peuvent cependant être rédigés en français.
